# 女娲 · Skill造人术（宏观研究员版）

> 基于 [nuwa-investor-skill](https://github.com/DZH123553/SKILL/tree/main/nuwa-investor-skill) 的宏观与政策分析特化版本

## 概述

本项目是女娲 Skill 造人术的**宏观研究员版本**，专门用于蒸馏全球顶级宏观研究者、政策分析师和宏观交易员的思维框架与决策模式。与投资人版本相比，本版本在以下维度进行了深度特化：

**调研维度**聚焦宏观研究者的分析框架，重点覆盖：
- 宏观经济周期判断方法论（债务周期、货币政策、财政政策）
- 全球资本流动与地缘政治分析框架
- 大类资产配置逻辑（股票、债券、商品、货币、另类资产）
- 政策解读与预期差分析能力
- 历史类比与跨周期比较思维

**心智模型提炼方向**从投资决策改为宏观分析相关的思维模型：
- 经济周期定位框架（Dalio 的债务周期、Marks 的周期理论）
- 政策传导机制分析（Pozsar 的货币体系、Tooze 的财政国家）
- 跨市场关联分析（Druckenmiller 的宏观交易、Soros 的反身性）
- 尾部风险识别（Bessent 的政策对冲、Alden 的能源-货币体系）
- 中国宏观经济特殊性（付鹏的资产负债表衰退、洪灏的周期定位）

**决策启发式方向**聚焦宏观判断准则：
- 经济处于周期的哪个位置？
- 政策方向与市场预期差多少？
- 资金流向哪里？为什么？
- 什么因素被低估了？什么被高估了？
- 历史类比中，这次哪里不同？

**输入源调整**重点纳入：
- 公开发表的宏观研究报告、年度展望
- 政策演讲、国会/议会证词、央行会议发言
- 播客访谈、电视评论、X/Twitter 宏观分析
- 著作与学术论文中的核心框架

## 核心能力：双轨 Agentic Protocol

本版本在女娲原有 Agentic Protocol 基础上，引入**双轨自主分析**：

- **Track A — 正向研究**：此人会关注的核心驱动因素、政策信号、资金流向
- **Track B — 反向分析（Contrarian Analysis）**：自主收集最能推翻主流共识的事实与观点

Skill 激活后评估具体宏观议题时，默认**同步**启动 A/B 双轨，通过 Steelman 反方三问、共识盲区映射、历史反例清单等机制避免确认偏误，最终以此人的**质疑风格**输出平衡的宏观判断。

**触发关键词**：用户说"挑毛病 / red team / 给我最坏情况 / 市场共识错在哪里 / 反方观点"时，Track B 将主导输出。

## 与原版女娲的兼容性

本版本保持了女娲原有的整体结构，包括四阶段蒸馏流程（采集 → 验证 → 构建 → 质检）和五层蒸馏体系（表达DNA、心智模型、决策启发式、反模式、诚实边界）。
