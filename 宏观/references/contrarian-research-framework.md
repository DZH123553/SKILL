# 反向分析方法论（Contrarian Analysis）

> 宏观版 Skill 在运行时评估宏观议题时必须执行的"反面分析"轨道

## 核心原则

1. 主动寻找不同意见，不是等它撞上来
2. 质量优先：过滤噪音，只采纳可追溯的独立来源
3. Steelman：把反方论点建构到最强版本
4. 对齐人设：通过"此研究者会怎么看"的滤镜输出

## 反向分析6维度

### 1. 数据与统计侧风险
- 官方数据是否可靠？季节性调整是否合理？
- 有没有幸存者偏差或结构性扭曲？
- **红旗**：数据修订方向一致、样本覆盖偏差、统计方法改变

### 2. 政策传导风险
- 政策的二阶、三阶效应是什么？
- 历史上有无类似政策导致反效果？
- **红旗**：政策目标与工具错配、传导机制被阻断

### 3. 共识盲区
- 市场主流叙事遗漏了什么变量？
- 哪些假设在 stress test 下会崩溃？
- **红旗**：共识过度拥挤、仓位与叙事背离

### 4. 历史反例
- 历史上类似判断有多少次被证伪？
- "这次不一样"的代价是什么？
- **红旗**：过度依赖单一历史类比、忽视结构性变化

### 5. 尾部风险
- 什么黑天鹅/灰天鹅被定价错误？
- 相关性在危机中如何变化？
- **红旗**：波动率微笑异常、信用利差过度压缩

### 6. 地缘政治/结构性突变
- 地缘冲突升级的概率和路径？
- 供应链/能源/技术的结构性断点？
- **红旗**：单点依赖、不可逆趋势被忽视

## 信息源优先级

| 级别 | 来源 | 用途 |
|------|------|------|
| S | 央行/财政部原始数据、政策文件 | 硬事实 |
| A | 独立研究机构、看空报告 | 结构性质疑 |
| A | 历史档案、学术复盘 | 历史反例 |
| B | 市场参与者访谈、内部人士观点 | 非官方视角 |
| B | X/Twitter 独立分析师 | 实时反方 |
| C | 主流媒体评论 | 共识风向（反向指标） |
