# 宏观研究员思维框架提炼方法论

> 从原始信息到可运行的宏观研究心智模型

## 一、调研 Input 维度（宏观特化）

1. **宏观经济周期判断框架**：
   - 如何判断经济所处周期位置？
   - 使用哪些领先/同步/滞后指标？
   - 对债务周期、信贷周期的核心观点？

2. **政策分析框架**：
   - 如何分析央行货币政策（利率、QE/QT、前瞻指引）？
   - 如何分析财政政策（赤字、债务、基建、税收）？
   - 政策传导机制怎么看？

3. **大类资产配置逻辑**：
   - 在不同周期阶段偏好什么资产？
   - 跨市场关联分析框架？
   - 对汇率、利率、信用利差的核心观点？

4. **地缘政治与结构性因素**：
   - 如何看待全球资本流动？
   - 对去全球化、供应链重构、能源转型的判断？
   - 大国博弈（中美欧）的宏观含义？

5. **历史类比与跨周期比较**：
   - 最常被引用的历史案例？
   - "这次不一样"还是"这次一样"的判断标准？

6. **反模式与质疑素材**：
   - 公开批评过的市场共识？
   - 判断失误的案例及反思？
   - 质疑句式与口头禅？

## 二、心智模型识别的三重验证

1. **跨周期复现**：同一框架在至少2个不同经济周期中出现过
2. **有预测力**：用这个模型可以推断此人对新政策的反应
3. **有区分度**：不是所有宏观研究者都这样看

## 三、表达DNA量化

| 维度 | 测量方式 |
|------|---------|
| 平均句长 | 字数/句数 |
| 数据引用密度 | 具体数字/千字 |
| 历史类比密度 | 历史案例/千字 |
| 条件句比例 | "如果...那么..."/总句数 |
| 确定性语气 | "必然"/"可能"/"也许"比例 |
| 学术术语密度 | 专业术语频率 |

**风格标签**：正式←→口语 / 抽象←→具体 / 谨慎←→断言 / 学术←→通俗 / 数据驱动←→叙事驱动
