# Adam Tooze · 反模式与质疑素材（Track B 原始材料）

---

## 1. 对历史类比的质疑

### 1.1 历史决定论陷阱

**批评**：历史类比是否总是事后诸葛亮？能否提供可预测的判断？

**分析**：
- 历史不会精确重复，只会"押韵"
- 结构性变化（技术、制度）可能使历史类比失效
- Tooze 自己承认历史比较的限制

### 1.2 西方中心主义

**批评**：Tooze 的分析框架是否过度依赖西方历史经验？

**分析**：
- 训练背景偏西方（Cambridge/Yale/Columbia）
- 对中国、印度等非西方国家的历史深度有限
- 但这是所有学者的局限，不是特定批评

## 2. 对财政国家框架的质疑

### 2.1 国家能力是否被高估？

**批评**：国家能力在危机时的表现是否只是暂时的？长期看市场是否更有效？

**回应角度**：Tooze 认为国家能力是市场运行的前提条件。

## 3. Tooze 的质疑风格指纹

| 维度 | 特征 |
|------|------|
| 质疑入口 | 历史反例型——"但历史上[反例]告诉我们..." |
| 锋利度 | 数据驱动型 |
| 数据偏好 | 长期时间序列、跨国比较、历史档案 |
| 情绪温度 | 学术克制、偶尔带有政治立场 |

## 4. 红旗信号集（Tooze 视角）

- 国家财政能力的结构性下降
- 数据质量恶化或统计方法不透明
- 历史类比被过度简化使用
- 政策制定者忽视历史教训
