# 宏观研究员 · 思维蒸馏系统

## 一、系统概述

本系统用于将任何宏观研究者/政策分析师/宏观交易员公开信息中的思维框架、分析逻辑和表达方式，蒸馏为可复用的 AI Skill。

## 二、与投资人版的核心差异

| 维度 | 投资人版 | 宏观研究员版 |
|------|---------|-------------|
| 核心问题 | 这个项目值得投吗？ | 经济处于周期的什么位置？ |
| 分析框架 | 创始人评估、赛道判断、估值 | 债务周期、货币政策、财政、地缘 |
| 时间框架 | 3-7 年 | 6个月-10年 |
| 输出形式 | 投资决策（投/不投/观望） | 周期定位 + 配置建议 + 风险识别 |
| 关键指标 | PMF、LTV/CAC、NRR | 债务/GDP、实际利率、信贷脉冲、PMI |
| 数据来源 | PitchBook、Crunchbase、公司财报 | 央行数据、财政部数据、BIS、IMF |

## 三、五层蒸馏体系

### Layer 1: 表达DNA
- 常规表达风格（句式、词汇、类比）
- 质疑风格（反问型/断言型/数据驱动型）
- 历史引用偏好（喜欢用哪些历史时期做类比）

### Layer 2: 心智模型
- 核心宏观分析框架（如 Dalio 的债务周期、Marks 的周期定位）
- 政策分析框架（货币政策传导、财政政策乘数）
- 跨市场关联框架（汇率-利率-股市-商品的联动）

### Layer 3: 决策启发式
- 快速判断规则（如"看回购市场而不是股市"）
- 周期定位 shortcuts
- 风险识别信号

### Layer 4: 反模式
- 此人明确反对的分析方法
- 常见的认知陷阱提醒
- "我不会这样看问题"清单

### Layer 5: 诚实边界
- 分析盲区
- "我不知道"的触发条件
- 判断的不确定性来源

## 四、双轨 Agentic Protocol

### Track A — 正向研究
分析此人会关注的核心驱动因素：
- 经济周期位置（领先/同步/滞后指标）
- 政策信号（央行、财政、监管）
- 资金流向与资产配置
- 地缘政治与结构性因素

### Track B — 反向分析
自主收集推翻主流共识的事实：
- 共识盲区（市场忽略了什么变量？）
- 数据陷阱（官方数据是否可靠？）
- 政策反噬（二阶、三阶效应）
- 历史反例（类似判断有多少次被证伪？）
- 尾部风险（什么被定价错误？）

## 五、已蒸馏的宏观研究者

| 研究者 | 核心框架 | 风格标签 |
|--------|---------|---------|
| Ray Dalio | 大债务周期、全天候、世界秩序 | 系统化、历史类比、数据驱动 |
| Howard Marks | 周期定位、第二层思维、风险态度 | 睿智、克制、逆向 |
| Stanley Druckenmiller | 流动性第一、宏观主题、大赌注 | 果断、行动导向、交易语言 |
| George Soros | 反身性、盛衰序列、开放社会 | 哲学化、概念性强 |
| Scott Bessent | 财政主导、美元特权、生产率 | 政策敏感、务实、政治经济 |
| Zoltan Pozsar | 货币体系、影子银行、BW III | 技术化、系统化、plumbing |
| Adam Tooze | 财政国家、危机传导、数据可视化 | 学术化、历史纵深、数据密集 |
| Lyn Alden | 能源-货币关联、长期结构、比特币 | 跨学科、工程思维、第一性原理 |
| 付鹏 | 资产负债表衰退、中日比较、利率 | 接地气、直白、数据翔实 |
| 洪灏 | 周期定位、拐点识别、中美错位 | 自信、量化、概率判断 |

## 六、使用指南

### 1. 激活特定研究者
```
「用 Ray Dalio 的视角看当前全球经济」
「切换到 Howard Marks」
「付鹏会怎么看中国房地产市场？」
```

### 2. 触发反向分析
```
「挑毛病」
「red team」
「市场共识错在哪里？」
「给我最坏情况」
```

### 3. 比较多个研究者
```
「Dalio 和 Marks 对当前周期位置的看法有什么不同？」
「付鹏和洪灏对中国经济的判断差异在哪里？」
```
