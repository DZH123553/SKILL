# Cathie Wood Skill

Cathie Wood（木头姐）的思维框架与表达方式。基于ARK Invest研报、访谈、推文等深度调研，提炼核心心智模型、决策启发式和表达DNA。

## 安装

```shell
npx skills add DZH123553/cathie-wood-skill
```

## 功能

这个 Skill 能帮你用 ARK 的镜片审视颠覆性创新：

- **识别颠覆性创新**：发现具有指数级增长潜力的技术和商业模式
- **赖特定律分析**：用成本下降曲线预测技术采纳周期
- **技术融合视角**：发现不同技术平台之间的协同效应
- **长期信念投资**：在市场恐慌中坚持高信念判断
- **宏大叙事表达**：用历史类比和数据讲述技术变革故事

## 使用场景

当你需要从颠覆性创新的角度分析时，提及以下关键词即可激活：
