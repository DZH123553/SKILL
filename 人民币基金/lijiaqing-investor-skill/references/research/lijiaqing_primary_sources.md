# 李家庆 / 君联资本 — 一手来源清单

## 一、君联资本官方来源

### 1. 君联资本官网 — 投资案例与团队介绍
- **URL**: https://www.legendcapital.com.cn/
- **类型**: 官方一手
- **内容**: 君联资本投资案例库（覆盖TMT、医疗健康、先进制造等）、管理团队介绍、基金概况
- **价值**: 了解君联 portfolio 全貌、管理规模（约700亿人民币+美元，20余支基金）、投资领域分布
- **关键提取**: 智冉医疗、百图生科、深睿医疗、耀速科技、派格医药、信达生物、贝瑞和康等案例的官方描述

### 2. 君联资本微信公众号 — 投资策略与动态
- **类型**: 官方一手
- **内容**: 君联资本官方公众号发布的投资公告、被投企业动态、行业观点文章
- **价值**: 获取最新投资动向、投后管理逻辑、行业趋势判断
- **关键提取**: "从Hunter到Farmer"转型的系列论述、新质生产力投资方向、医疗投资方法论

### 3. 君联资本年度报告 / LP 通讯
- **类型**: 官方一手（非完全公开）
- **内容**: 基金业绩、投资策略更新、宏观经济判断
- **价值**: 了解君联内部对投资环境的真实判断（通过公开引述或行业会议披露）
- **关键提取**: 管理规模数据（~700亿，899个融资事件，638家公司）、双币基金运作逻辑

### 4. 联想控股年报 — 君联资本板块
- **URL**: https://www.legendholdings.com.cn/
- **类型**: 上市公司关联一手
- **内容**: 联想控股作为母公司对君联资本的业绩披露、战略定位描述
- **价值**: 从母公司视角理解君联在联想系生态中的角色和约束
- **关键提取**: 君联资本对联想控股利润贡献、关联交易披露、战略协同描述
